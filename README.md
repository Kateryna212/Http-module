# Практична робота №4: Нативний HTTP у Node.js

Цей репозиторій містить розв'язання вправ з навчального модуля `eu-node-basics-workshop` (Practice 4), присвяченого побудові HTTP-сервера без використання сторонніх фреймворків.

## Виконані вправи:
* **4.1 ROOT ROUTE** — створення базового сервера та обробка шляху `/`.
* **4.2 TIME ROUTE** — робота з JSON та форматом ISO.
* **4.3 ECHO ROUTE** — отримання параметрів з Query String.
* **4.4 SUM ROUTE** — валідація вхідних даних та обчислення суми.
* **4.5 NOT FOUND ROUTE** — коректна обробка неіснуючих маршрутів (статус 404).

## Технологічний стек:
* Node.js v24.15.0 LTS (або вище)
* Вбудований модуль `http`
* Вбудований модуль `url`
